# User Management Dashboard

A modern **Admin User Management Dashboard** built using **HTML, CSS, and JavaScript**.

This project was created as part of an internship assignment to demonstrate CRUD operations, form validation, and responsive UI design with a premium dashboard-style interface.

---

## Live Demo

🚀 **Project Live Link:** https://notebook-demo01.vercel.app/

---

## Features

### Core Functionality

* **Display Users** in an admin-style table
* **Add User** with form submission
* **Edit User** details
* **Delete User** permanently
* **Search Users** by name or email

### Validation

* Email format validation using regex
* Submit button remains disabled until:

  * Name field is filled
  * Email is valid

### UI Enhancements

* Modern glassmorphism dashboard design
* Gradient background
* Responsive layout
* User avatars with initials
* Animated buttons and hover effects
* Live user counter
* Empty state display
* Toast notifications for:

  * Add user
  * Edit user
  * Delete user
* Delete confirmation popup
